# 博士后招聘 - 大连理工大学计算机科学与技术学院

## 项目简介

本项目包含了**大连理工大学计算机科学与技术学院**的**博士后招聘页面**，旨在招聘加入**张强教授**与**刘婵娟教授**课题组的博士后。研究方向涵盖智能决策、优化与控制，尤其专注于无人系统的相关领域。

## 研究方向

博士后研究方向包括但不限于以下内容：
- 智能博弈
- 强化学习
- 因果推理
- 多智能体系统
- 群体智能
- 无人系统
- 目标识别
- 协同控制
- 优化算法

## 福利待遇

1. 全口径工资25-30万元/年（不含科研提成和团队绩效）；
2. 团队可给予最高20万元绩效奖励；
3. 出站后，留连工作并认定为大连市青年才俊，可申请安家费30万元；
4. 出站后，留连工作并认定为“兴辽英才计划-博士后”，可获30万元奖励；
5. 优秀博士后出站推荐申请大连理工大学助理教授或副教授，可获安家费50-100万元；
6. 如入选国家、校级相应人才计划，在学校待遇基础上叠加资助；
7. 按照学校预聘师资进行管理，享有学校教职工同等福利待遇，如用餐补助、健康体检等；
8. 解决子女入托、入学，提供大连市最优质的基础教育资源；

*地方政策请以最新文件为准。

## 招聘条件

- 大连理工大学自1985年开始设立博士后科研流动站，是全国首批设站单位之一。
- 博士后团队近年来获奖项众多，并且一批博士后已经成为学校科研创新的主力军。自2019年以来，超过1/3的博士后留校任教，并承担多个重大科研课题。
- 博士后们的研究成果包括获得国家自然科学基金、国家社科基金等资助，累计研究经费超过7000万元。
